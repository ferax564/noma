Noma Cloud

Private workspace access

Noma Cloud is restricted to invited collaborators. Use the cloud access token to enter the app, then register a user only when you have an invitation code.

  • Cloud access uses a server-side HttpOnly session cookie.
  • New user registration requires an invitation code.
  • Existing user tokens can be imported during login.

Shared published artifacts still require their own share tokens. This page does not expose the app shell until access is verified.

Log in

Enter the cloud access token. Add your existing Noma user token if you want this browser to load your workspaces immediately.

Register with invitation

Create a new Noma Cloud user. Registration is blocked unless the invitation code matches the server-side code.
